What is OpenCV4Android?

Open CV (http://opencv.org/) is written in C++. It is the open source image recognition tool used on many platforms.

OpenCV4Android is a port to Android that adds a Java wrapper around the C++ code. Homepage:

http://opencv.org/platforms/android.html

Most of their documentation doesnt use Android Studio. That is because Eclipse supported the NDK in ways that Android Studio chooses NOT to do. There are more steps in Android Studio to do the setup and their documents never got updated. Oh well.

Overview of installing OpenCV4Android in Android Studio

  • Download latest supported OpenCV sdk for Android from OpenCV.org and decompress the zip file (already done).
  • Create an OpenCV module. From File New Import Module..., choose the sdk/java folder from the download.
  • Update the build.gradle file for the OpenCV module to update 4 fields to match the build.gradle file for your app module. (1) compileSdkVersion (2) buildToolsVersion (delete line) (3) minSdkVersion and (4) targetSdkVersion.
  • Add a module dependency right click on the app module. Select Open Module Settings, and select the Dependencies tab. Click + icon at bottom, choose Module Dependency and select the imported OpenCV module.
  • Copy the libs folder that is under sdk/native into Android Studio under app/src/main (easiest to do in Windows Explorer)
  • Rename the copied libs directory to jniLibs and we are done (from either Windows Explorer or Android Studio).

BTW my instructions are based on an answer to this Stackoverflow post:

http://stackoverflow.com/questions/17767557/how-to-use-opencv-in-android-studio-using-gradle-build-tool

More details are given for every step in the slides that follow. In the future the info above is all you will need.
